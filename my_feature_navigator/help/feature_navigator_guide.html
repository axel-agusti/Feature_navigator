Guía de Uso: Feature Navigator para QGIS

Pasos para utilizar el plugin Feature Navigator

El plugin Feature Navigator facilita la navegación entre entidades de una capa activa en QGIS, permitiendo desplazarse entre ellas utilizando los botones de "Anterior" y "Siguiente". A continuación, se describen los pasos para instalar y utilizar correctamente el plugin:

1. Instalación del plugin

  • Descargue e instale el plugin desde el repositorio de plugins de QGIS o desde el archivo comprimido si lo tiene disponible.
  • Una vez instalado, el plugin aparecerá en la barra de herramientas de QGIS como un icono con un botón de "Navegador de Entidades".

2. Activación del plugin

  • Haga clic en el icono de "Navegador de Entidades" en la barra de herramientas.
  • Esto abrirá una ventana flotante (Dock) que mostrará el número de la entidad seleccionada y el total de objetos disponibles.

3. Navegación entre entidades

  • En la ventana flotante, encontrará dos botones: "Siguiente" y "Anterior".
  • Haga clic en "Siguiente" para navegar hacia la siguiente entidad o en "Anterior" para volver a la entidad anterior.
  • El mapa se ajustará automáticamente para centrar y hacer zoom en la entidad seleccionada.

4. Uso de la capa activa

  • Para que el plugin funcione, debe tener una capa activa seleccionada en QGIS. La capa debe contener entidades (por ejemplo, puntos, líneas, polígonos).
  • El plugin utilizará las entidades de la capa activa para navegar entre ellas.

5. Personalización

  • Puede ajustar las dimensiones de la ventana flotante del plugin arrastrándola o redimensionándola según sus necesidades.
